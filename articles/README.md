# Articles Directory

This directory contains all Ethereum-related articles in the knowledge base.

## Structure

Each article should be a markdown file following the template provided in `../templates/article-template.md`.

## Naming Convention

Files should be named using lowercase with hyphens for spaces:
- `ethereum-merge-overview.md`
- `proof-of-stake-explained.md`
- `eip-1559-analysis.md`

## Organization

Articles can be organized into subdirectories by category:
- `protocol/` - EIPs, protocol upgrades, consensus mechanisms, execution layer
- `scaling/` - Layer 2, rollups, sharding
- `security/` - Audits, vulnerabilities, best practices
- `privacy/` - Privacy features, zero-knowledge proofs, privacy protocols
- `tools/` - Development tools, wallets, infrastructure
- `general/` - General information and overviews

Feel free to add new categories as needed.
